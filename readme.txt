------->> 舒尔特表简介 <<-------
舒尔特表（Schulte Grid）
作用：
    培养注意力集中、分配、控制能力；拓展视幅；加快视频；提高视觉的稳定性、辨别力、定向搜索能力。
    练习的时间越长，看表所需的时间会越短。随着练习的深入，眼球的末梢视觉能力提高，不仅初学者可以有效地拓展视幅，加快阅读节奏，锻炼眼睛快速认读；而且对于进入提高阶段之后，同时拓展纵横视幅，达到一目十行、一目一页非常有效。

使用方法
    眼睛距表30～35厘米，视点自然放在表的中心；
    在所有字符全部清晰入目的前提下，按顺序(1～9，A～I，汉字应先熟悉原文顺序)找全所有字符，注意不要顾此失彼，因找一个字符而对其他字符视而不见；
    每看完一个表，眼睛稍做休息，或闭目，或做眼保健操，不要过分疲劳；
    练习初期不考虑记忆因素。每天看10个表。

------->> 游戏说明 <<-------
    程序开始运行后，请求用户输入一个整数n，
    而后，程序生成一个随机的 n 行 n 列的舒尔特表并输出到屏幕上，同时游戏开始计时
    用户完成该表后，按下回车，结束计时，
    游戏计算并输出用户完成该表所用时间（以秒为单位）；
    本局游戏结束，进入下一局。

    每次运行程序，用户完成十张舒尔特表后程序结束。
    输入整数时，用户输入数字0表示提前终止程序；
    输入空行（也就是直接按下回车）程序使用上一张舒尔特表的n值，第一张舒尔特表的‘上一张’n值设为3；
    输入的开头和末尾的空格被忽略。
    其它任何输入均不可用，程序会要求用户重新输入，直到得到合法输入。

    程序结束运行前，会将用户完成各张舒尔特表所用时间保存至 ./resources/grades.json 文件中。
------->> 使用说明 <<-------
程序不需要安装，在命令行下直接运行。
Python 版本 2.7x。
不需要任何第三方类库。

要运行程序，从程序的根目录下，运行命令：
    > python Schulte.py
